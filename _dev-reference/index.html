vivianredpanda.github.io — Dev Reference

Interactive tools for positioning and tuning scene elements. Open via the Vite dev server (npm run dev) so image paths resolve correctly.

Interactive Editors

cozyroom-editor.html
Drag objects (kitchen, window, bookshelf, aquarium, plant) to set image-% positions. Sliders for VP_IMG_X, VP_IMG_Y, FACTOR, PERSP_DIST. Preview ratio slider simulates non-native viewports to test perspective rotation. Copy outputs OBJECTS array + constants.
CozyRoom.jsx
kitchen-editor.html
Drag jar images (jug, jar_1–jar_5, container) to set left/bottom canvas-pixel positions. Right edge resizes width. Sliders adjust labelTop, labelLeft, labelSize. Copy outputs CATEGORY_JARS array.
KitchenScene.jsx
study-book-editor.html
Drag numbered book hotspots to set top/left canvas-pixel positions. Right/bottom edges resize width/height. Add and remove books to match project count. Copy outputs BOOK_POSITIONS array.
StudyScene.jsx
ray-debug.html
Canvas tool for tuning the manta ray body animation. Live sliders for body strip corners (TL/TR/BR/BL) and wing anchor line (the pivot axis the flap rotates around). Values are normalized 0–1; copy back into RayCanvas.jsx.
RayCanvas.jsx
